## 🛠️ How to Run

Clone the repository and compile files based on their language:

```bash
git clone https://github.com/JGaurav26/LB_2025.git
cd LB_2025
```

#### Compile & Run (C)

```bash
# Compile
gcc path/to/file.c -o program
# Run
./program
```

#### Compile & Run (C++)
```bash
# Compile
g++ path/to/file.cpp -o program
# Run
./program
```

#### Compile & Run (Java)
```bash
# Compile
javac path/to/FileName.java
# Run
java -cp path/to FileName
```
